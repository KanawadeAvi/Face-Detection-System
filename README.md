Face Detection Login System Using Python face_recognition               

Run app.py file
Packages Requried
1. pip install face_recognition
2. pip install dlib
3. pip install cmake
4. pip install Flask

System Working
After  running Use landing on Index page it has two options
       1.Signup
       2.Login

MongoDB database is used to store images and Username.
